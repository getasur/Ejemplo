Tutorial para saber hacer una pagina web con html

Lo primero que se debe hacer para crear una pagina web con html sera escribir <!DOCTYPE html>, despues pondremos los siguientes comandos <html> y <body>. Los cuales se cierran con </html< y </body>. Los cuales iran escritos al final.



Despues pondremos un titulo con <title>, una vez escrito el titulo cerraremos con </title>y el titulo debera ir entre <head> y </head>



Una vez hecho esto pondremos un encabezado con el apartado <h1> y </h1> y los parrafos que queramos con el apartado <p> y </p>



Y con esto acabaria el pequeño tutorial
